# Jujutsu Sorcerer RPG - 2D Edition

A 2D pixel art action RPG built using Python and Pygame. Defeat cursed spirits, face off against a Special Grade Boss, and unleash your Domain Expansion!

## Features
- Smooth player movement and gravity-based jumping physics.
- Projectile combat mechanics.
- Infinite scrolling parallax background.
- Special Grade Boss fight unlocked at 100 points.
- Domain Expansion ultimate area-of-effect ability.
- Interactive Start, Game Over, and Win screens.
- Standalone executable build available.

## Controls
- **Left / Right Arrow Keys**: Move
- **Up Arrow Key**: Jump
- **Spacebar**: Shoot Cursed Energy
- **D**: Activate Domain Expansion (Costs 50 Score)
- **Enter**: Start / Restart / Play Again
